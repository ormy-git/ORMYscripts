## 🔍 SEO & Performance

Chaque page intègre :

- ✅ Balises `<title>` et `<meta description>` optimisées par page
- ✅ Balises **Open Graph** (LinkedIn, Facebook) et **Twitter Card**
- ✅ **Schema.org JSON-LD** (ProfessionalService, WebSite)
- ✅ `<link rel="canonical">` anti-duplication
- ✅ `sitemap.xml` soumis aux moteurs de recherche
- ✅ `robots.txt` configuré
- ✅ `rel="preconnect"` et `dns-prefetch` sur Google Fonts
- ✅ Favicon SVG inline (léger, vectoriel)
- ✅ Pages ultra-légères : **25 à 35 KB** par page

## 📦 Déploiement sur GitHub Pages

### 1️⃣ Créer le dépôt
```
Nom du dépôt : ormyscripts.github.io
Visibilité   : Public
```

### 2️⃣ Uploader les fichiers
Glissez tous les fichiers à la **racine** du dépôt (pas dans un sous-dossier) :
```
index.html
automatisations.html
prestations.html
tarifs.html
reseaux-sociaux.html
sitemap.xml
robots.txt
```

### 3️⃣ Activer GitHub Pages
```
Settings → Pages → Source : Deploy from branch → main → / (root) → Save
```

### 4️⃣ Accéder au site
⏱️ Après 1 à 2 minutes :
```
https://ormyscripts.github.io
```
